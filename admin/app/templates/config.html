Python (wavelink / discord.py)

Configure Wavelink to connect to this node in your Discord bot's setup hook.

import wavelink

async def setup_hook():
    node = wavelink.Node(
        uri="http://YOUR_SERVER_IP:2333",
        password="{{ password }}"
    )
    await wavelink.Pool.connect(nodes=[node], client=bot, cache_capacity=100)

Rust (lavalink-rs / serenity)

Connect to the node using lavalink-rs.

use lavalink_rs::{LavalinkClient, model::events};

let node = NodeBuilder::new("Node 1")
    .hostname("YOUR_SERVER_IP:2333")
    .password("{{ password }}")
    .build()?;

let client = LavalinkClient::builder(bot_id)
    .add_node(node)
    .build(events)
    .await?;

Ready-made Discord Bots

If you don't want to code your own bot, you can host these popular self-hosted bots and point them to your node:

  • JMusicBot - Java, highly popular and stable.
  • Discord-MusicBot - Node.js based feature-rich bot.
  • Muse - TypeScript/Discord.js bot.

Simply enter YOUR_SERVER_IP:2333 and password {{ password }} into their respective config files!
